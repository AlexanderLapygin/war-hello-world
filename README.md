---

## Build

```sh
mvnw clean package
```

## Run

```sh
java -cp target/hello-world-1.0-SNAPSHOT.jar com.example.App
```

Deploy war to Tomcat, for example by manually copying to webapps directory.
Take a look at http://localhost:8080/hello.
